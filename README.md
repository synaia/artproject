### Welcome

Interactive proposal for the Bienal Art Event

Steps:

```shell
clone this repo
conda create -n artproject_env python=3.10 -y
conda activate artproject_env
```


On root project directory.
```shell
pip install -r min_reqs.txt
```

Run server
```shell
./runserver.sh
```

On client project directory.
```shell
npm install
```

Run client
```shell
npm run start
```
